Heron Quick Reference
  • File layout: - Module declaration followed by type declarations.
  • Module: - module MyModule { imports { ... } fields { ... } methods { ... } }
  • Class: - class MyClass { inherits { ... } fields { ... } methods { ... } }
  • Interface: - interface MyInterface { inherits { ... } methods { ... } }
  • Enum: - enum MyEnum { fields { valueName; } }
  • Control flow: - foreach, for, while, if.
  • List operators: - map, select, reduce, accumulate.
  • Arithmetic operators: - +, -, *, /, %, ++.
  • Comparison operators: - ==, !=, <, <=, >, >.
  • Boolean operators - &&, ||
  • Heron types: Int, Char, String, Float, Bool, List, Seq, Function.
  • Integer ranges: [1..100]
  • Strings: "hello world"
  • Verbatim strings: @"hello world"
  • Characters: 'a'
  • Booleans: true
  • Records: record (a:Int, b:String) { 42, "forty-two" }
  • Tables: table (a:Int, b:String) { one, "one"; two, "two"; }
  • Functions: - function (a:Int) { WriteLine(a.ToString()); }
